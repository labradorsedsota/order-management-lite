### Pre-flight 标准流程（每条测试前执行）

```bash
# 1. 打开目标页面（强制重置页面状态）
open -a "Google Chrome" "https://labradorsedsota.github.io/order-management-lite/"
sleep 3

# 2. 窗口最大化（动态获取屏幕实际尺寸，不写死数值）

tell application "Finder"
    set _b to bounds of window of desktop

    end tell
end tell'
```

### localStorage 清空脚本（L3.2/L3.3/L3.6 专用）

```bash
osascript -e '
tell application "Google Chrome"
    tell active tab of front window
        execute javascript "localStorage.clear(); location.reload();"
    end tell
end tell'
sleep 2
```
