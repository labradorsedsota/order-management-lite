# CHANGELOG — OMS 轻量级订单管理系统

## v1.0 (2026-04-10)

**初始发布**

- 仪表盘：4 张统计卡片（今日订单、待处理、本月销售额、总订单数），实时刷新
- 订单管理：新建（多商品）、编辑、删除、状态全流程推进（待确认→已确认→生产中→已发货→已完成）
- 客户配置：完整 CRUD
- 商品配置：完整 CRUD
- 筛选功能：按状态、客户、日期范围筛选 + 组合筛选 + 重置
- 数据联动：客户/商品下拉自动带入单价，金额实时计算
- 数据持久化：localStorage 存储 + 首次加载种子数据
- 容错处理：localStorage 数据损坏时静默降级为空数据
- 表单验证：必填项、单价>0、至少一个商品
- 删除保护：二次确认对话框
- 导出按钮：Toast 提示（预留功能）
- 视觉风格：Apple 设计规范（#F5F5F7 背景、#007AFF 主色、圆角卡片、系统字体栈）
